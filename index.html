Programa de Contraseña Segura

Genera contraseñas seguras a partir de tus palabras, o valida si una contraseña cumple los requisitos de seguridad.

Generador

Crea contraseñas seguras usando tus propias palabras como base.

Validador

Verifica si tu contraseña cumple los requisitos de seguridad.

Generador de contraseñas

Escribe entre 1 y 3 palabras. El programa las usará para crear tus contraseñas.

Validador de contraseñas

Escribe una contraseña para analizar su nivel de seguridad.

Nivel de seguridad —
Mínimo 12 caracteres
Al menos una letra mayúscula
Al menos una letra minúscula
Al menos un número
Al menos un símbolo especial
Contraseña segura. Cumple con todos los requisitos recomendados.
